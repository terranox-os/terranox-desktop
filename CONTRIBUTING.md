# Contributing

Follow [TerranoxOS contributing norms](https://github.com/terranox-os/terranox-os/blob/develop/CONTRIBUTING.md): **Conventional Commits** and **`git commit -s`** (DCO).

Match the **SPDX license** of the crate you edit (`GPL-2.0-only` vs `Apache-2.0`).
